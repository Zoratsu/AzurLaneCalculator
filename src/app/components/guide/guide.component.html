Instructions

For Equipment Calculator: (Work In Progress)

  1. Go to the navigation bar
  2. Click the menu
  3. Click Equipment option
  4. Select the equipment type
  5. Select a nation to filter equipment if needed
  6. Select the equipment
  7. Select the tier
  8. Done

Editing Gun Stats

  1. Click Stats tab
  2. Click tab you want to edit, "Base" are the gun Stats and "Modifier" are the ammo modifiers
    1. The calculator uses Azur Lane Wiki as basis for format of values
  3. Edit stats as you need/want
  4. When done, click Update

Viewing results

  1. Click Calculation tab

For Ship Calculator: (Work In Progress)

  1. Go to the navigation bar
  2. Click the menu
  3. Click Ship option
  4. Select the hull type
  5. Select a nation to filter ships if needed
  6. Select the ship
  7. Select the level
  8. Done

Modifying Ship Stats

  • Modifying Slots Efficiency
    1. Select "Ship Name" Stats tab
    2. Select Efficiencies tab
    3. Edit stats as you need/want
    4. When done, click Update
  • Equipping a Equipment
    1. Select Slot tab you want
      • The calculator uses Azur Lane Wiki as basis for format of values
    2. Follow Equipment Calculator to select equipment
    3. When done, click Equip
  • Modifying a Slot
    1. Select Slot tab you want to edit
      • The calculator uses Azur Lane Wiki as basis for format of values
    2. Edit stats as you need/want
    3. When done, click Update

This mode uses a reduced Equipment Calculator in the Ship Slots, so check it's instructions if you need help

Instructions

For Equipment Calculator: (Work In Progress)

  1. Go to the navigation bar
  2. Click the menu
  3. Click Equipment option
  4. Select the equipment type
  5. Select a nation to filter if needed
  6. Select equipment
  7. Select tier

You can edit any of the stats, but FP/AA/AVI are not used in this mode so let them in 0 if editing gun stats After any changes to stats, click Update button to redo the calculations

For Ship Calculator: (Work In Progress)

  1. Go to the navigation bar
  2. Click the menu
  3. Click Ship option
  4. Select the hull type
  5. Select a nation to filter if needed
  6. Select ship
  7. Select stat level

This mode uses Equipment Calculator too, so check it's instructions After any changes to stats, click Update button to redo the calculations
